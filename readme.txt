横道图功能研究；


功能介绍：
	采用三层架构开发，涉及技术有Ajax,Jquery等！
	系统采用jquery easyui 里面的datagrid进行增删改查绑定数据！
	增加信息：实现验证不为空，邮箱格式验证；
	删除信息：实现对选中删除信息进行判断，全选删除，删除成功信息等功能！
	修改信息：对编辑用户进行信息修改，并弹框提示！
	查询信息：可根据条件进行查询！
	分页功能：翻页，页面显示行数及总页数，可根据不同条件显示信息的行数来决定页数！


开发环境：Visual Studio 2010；数据库：SQL2008
